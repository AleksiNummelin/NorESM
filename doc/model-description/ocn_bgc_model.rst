The ocean BGC model
=======================================

The ocean biogeochemistry (BGC) component **iHAMOCC** (isopycnic coordinate HAMburg Ocean Carbon Cycle model) is an updated version of the ocean biogeochemistry module used in NorESM1. Details on the updates and improvements of the ocean biogeochemical component of NorESM2 are provided in Tjiputra et al. (2020).
